## Features

- Typescript
- NextJS
- Serverless
- Deploy to Vercel

## Getting Started

#### Install dependencies
```sh
yarn install
```

#### Build the app
```sh
yarn bot:build
```

#### Start the app
```sh
yarn bot:build
```
